Kiddion's Modest External Menu

Popular external menu for GTA Online.

Usage

Start GTA5 and wait until the game has fully loaded.
Start the mod
Read the disclaimer

Default keys include:
○ F5 to show/hide the menu,
○ Numpad_0 to go back,
○ Numpad_8 and Numpad_2 to navigate up/down through the menu options.
○ Numpad_4 and Numpad_6 to decrease/increase the current value.
○ Numpad_5 to activate an option, toggle its value or applying any changed setting.
