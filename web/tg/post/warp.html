WARP

Cloudflare Warp is a TOR implementation by Cloudflare. WARP encrypts all the traffic from your device and send it directly to Cloudflare’s nearest bridge, and then to destination server. It works on mobile and PC.

Main benefit of using WARP is while it acts like VPN, it does not make your connection speed slower. WARP will help you hide your traffic from ISP and gain access to blocked resourses, protect your activity in public WIFI networks, hide your real IP (your IP still be representing your country because WARP will connect you to nearest bridge) and protect it from (D)DoS attacks. However WARP is not designed for being anonymous, Cloudflare logs WARP usage.

Usage

○ Download WireGuard client.

○ Download zip archive with .conf file and unpack it.

○ Import .conf file into WireGuard.

○ Activate connection and test it by pressing Test WARP connection button above. Using DNS over WARP parameter should be Yes.
